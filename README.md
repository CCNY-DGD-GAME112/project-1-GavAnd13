Hello, this is a silly clicker game I made for my game design class. You're on a submarine, but you're running out of air and need to pump more in by clicking the pump button. You will passively lose air at an increasing rate as the game goes on. How long can you survive?

There are four upgrades in the game:
A pump upgrade increases the amount of air generated per click.
An auto pump upgrade mimics you clicking every second.
A passive pump upgrade generates one oxygen per second.
A capacity upgrade increases how much air you can store.

There isn't really a lot of game balancing right now so just have fun with it.
